# Top Panel for Windows

A lightweight macOS-like top panel for Windows built with C# and .NET to
enhance desktop productivity.

## Preview

![Top Panel](screenshots/app.png)

## Description

Top Panel for Windows is a desktop utility application that provides a
macOS-style top panel on Windows.\
It is designed to improve user experience by offering quick access to
system information, shortcuts, and customizable features.

This project is developed for learning C# desktop development and
building a professional portfolio.

## Features

-   macOS-like top panel
-   Always on top panel
-   System time display
-   Application shortcuts
-   Lightweight and fast
-   Simple and clean UI
-   Customizable layout (future)

## Technologies Used

## Installation

### 1. Clone repository

``` bash
git clone https://github.com/your-username/top-panel-for-windows.git
```

### 2. Open project

Open with Visual Studio Community

    TopPanel.sln

### 3. Run project

Press

    F5

or

    Start Debugging

## Build

In Visual Studio

    Build → Build Solution

Output will be in

    bin/Debug
